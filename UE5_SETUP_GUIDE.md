# UE5 Setup Guide for The Great War of Land

## Prerequisites
1. **Download Unreal Engine 5**  
   Go to the [Epic Games website](https://www.unrealengine.com/en-US/download) to download the Epic Games Launcher.  
   Install it and then use it to install Unreal Engine 5.

2. **Create an Epic Games Account**  
   If you donâ€™t have one, sign up at the [Epic Games signup page](https://www.epicgames.com/account/signup).

3. **Recommended System Requirements**  
   Ensure your system meets the following minimum specs:  
   - **Operating System:** Windows 10 64-bit or macOS 10.14 or later  
   - **Processor:** Intel or AMD multi-core processor, 2.5 GHz or faster  
   - **RAM:** 16 GB or more  
   - **Graphics Card:** NVIDIA GeForce  GTX 770 or AMD Radeon  RX 470 or better  
   - **DirectX:** Version 11 or 12

## Project Setup
1. **Open Unreal Engine 5**  
   Launch the Epic Games Launcher, navigate to the Library tab, and click on the 'Launch' button under Unreal Engine 5.

2. **Create a New Project**  
   - Click on the 'New Project' tab.  
   - Select the Template you want to use (First Person, Third Person, etc.).  
   - Set the project settings, ensuring "With Starter Content" is selected, and then name your project.

3. **Get the Project Files from GitHub**  
   Open the terminal or command prompt and run the following command:
   ```bash
   git clone https://github.com/AkiUse306/the-great-war-of-land.git
   ```

4. **Open the Project**  
   - Navigate to the directory where you cloned the project.  
   - Double-click the project file (*.uproject) to open it in Unreal Engine 5.

## Asset Management
1. **Import Assets**  
   - Use the Content Browser to import any additional assets needed for the project.  
   - Make sure all assets are properly organized in folders.

2. **Link External Libraries**  
   - If you are using external plugins or libraries, ensure they are correctly installed and linked within the project settings.

## Testing
1. **Run the Project**  
   - Click on the 'Play' button to test your project.  
   - Debug any issues that arise during gameplay.
